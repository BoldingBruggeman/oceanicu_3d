Comprehensive ocean model validation platform developed for systematic evaluation of numerical ocean models.

Purpose#

This dashboard provides:

  • Standardized Validation: Consistent metrics across all simulations
  • Comprehensive Analysis: Multiple validation types and data sources
  • Transparent Results: Full statistics and plots for all experiments
  • Scenario Evaluation: Future projection analysis without observations

Validation Types#

Standard Validation#

Comparison of model output against observations:

  • Satellite products (SST, SSS, SSH)
  • In-situ profiles (Argo floats)
  • Tide gauge data

Tidal Analysis#

Harmonic analysis of sea surface elevation:

  • Constituent extraction and validation
  • Satellite product comparison (FES2014, TPXO9)
  • Coastal station validation

Scenario Analysis#

Future climate projections:

  • Trend analysis
  • Multi-scenario comparison
  • Period-to-period changes
  • Extreme value analysis

Methodology#

Models are evaluated using multiple statistical metrics:

  • RMSE: Root Mean Square Error
  • Bias: Mean difference (Model - Observation)
  • MAE: Mean Absolute Error
  • Correlation: Pearson correlation coefficient

Rankings combine these metrics using weighted scores to identify best-performing configurations.

Data Sources#

All validation uses publicly available data from:

  • Satellite missions (NOAA, ESA, NASA)
  • International programs (Argo, GESLA)
  • Reanalysis products (ERA5)

See Data Sources for complete list.
