Atmospheric Forcing — Bias Correction

April 11, 2026 · Ocean Modeling Team
Table of Contents
  • SSP Scenario Reference

Diagnostics for the bias correction of CMIP6 atmospheric forcing fields applied to all regional ocean simulations. Results are scenario-specific and valid across all model domains.

Bias correction is applied using quantile delta mapping (QDM) calibrated against ERA5 reanalysis over the historical period. The plots below show calibration performance and projected trends under each SSP scenario.
