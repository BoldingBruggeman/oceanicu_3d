Methodology#

Harmonic Analysis#

Tidal constituents extracted using UTide harmonic analysis package. Analysis performed on high-frequency model output (10-15 min resolution) over a minimum period of 60 days.

Satellite Comparison#

Model results compared with satellite-derived tidal products:

  • FES2014: Finite Element Solution 2014 (1/16° resolution)
  • TPXO9: OSU Tidal Prediction Software (1/30° resolution)

Satellite data regridded to model grid using conservative interpolation.
