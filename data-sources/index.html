Satellite products, in-situ observations, reanalysis data, and tidal atlases used for model validation. All sources are publicly available. The data-loader command provides a unified interface for local files, Copernicus Marine, CMIP6, Argo, ICES/EMODnet, and GLODAP data.

data-loader /path/to/file.nc --info          # inspect a local file
data-loader --cmems-describe <DATASET_ID>    # describe a Copernicus dataset

YAML configuration#

Any download can be described in a YAML block and passed to the validation tools — no separate download step required. The observations: block inside every analysis config accepts the same keys:

observations:
  source: copernicus                          # copernicus | cmip6 | isimip | auto
  dataset_id: cmems_obs-sst_atl_phy_nrt_l3s_P1D-m
  variables: [sea_surface_temperature]
  domain:
    lat_min: 50
    lat_max: 65
    lon_min: -15
    lon_max: 15
  time_range: [2020-01-01, 2020-12-31]

For a local file the same block simply points to a path:

observations:
  path: /data/CCI-SST/sst_2020.nc
  variable: analysed_sst

Programmatic access#

DataLoader.load_from_config() accepts the parsed YAML dict directly, making it straightforward to integrate into notebooks or custom scripts:

import yaml
from data_loader import DataLoader

with open("config/gridded_2d_validation_surface.yaml") as f:
    cfg = yaml.safe_load(f)

ds = DataLoader.load_from_config(cfg, section="observations")

Gridded Climatologies#

World Ocean Atlas (WOA23)#

Quality-controlled temperature and salinity climatologies on a global 0.25° grid. Used as reference fields for gridded 3D validation.

  • Source: NOAA NCEI
  • Variables: Temperature, Salinity
  • Resolution: 0.25° (monthly climatology)
# WOA23 files are referenced directly in the config
gridded-3d-validation --config config/gridded_3d_validation.yaml

CMIP6 Reanalysis / Model Output#

Global ocean model output from the PANGEO CMIP6 catalog.

  • Catalog: PANGEO CMIP6
data-loader --cmip6 CESM2/historical/tos/Omon/r1i1p1f1 \
    --domain -5. 15. 50. 65. \
    --time-range 1990-01-01 1990-12-31 \
    --output cesm2_tos.nc

In-Situ Profiles#

Argo Floats#

Freely drifting profiling floats measuring temperature and salinity. Downloaded on-demand from the Ifremer ERDDAP service.

  • Source: Argo GDAC — Ifremer ERDDAP
  • Variables: Temperature, Salinity (0–2000 m)
  • Coverage: Global
argo-profiles --config config/argo_profiles.yaml

ICES / EMODnet Chemistry Cruise CTD Profiles#

Ship-based CTD profiles from the ICES and EMODnet Chemistry databases. Downloaded on-demand from the EMODnet Chemistry ERDDAP service.

  • Source: EMODnet Chemistry ERDDAP
  • Variables: Temperature, Salinity (and biogeochemical where available)
  • Coverage: European seas and North Atlantic
cruise-ctd-profiles --config config/cruise_ctd_profiles.yaml

GLODAP v2.2023#

Quality-controlled bottle data from global research cruises (1957–2021). Includes physical and biogeochemical variables.

  • Source: GLODAP
  • Archive: NOAA NCEI / OCADS
  • Variables: Temperature, Salinity, DIC, Alkalinity, Oxygen, Nutrients
  • Coverage: Global; ~1.3 million bottle samples
# Data file downloaded automatically on first run (~110 MB ZIP)
glodap-profiles --config config/glodap_profiles.yaml

Tidal Products#

GESLA v3 — Tide Gauges#

Global Extreme Sea Level Analysis: high-frequency tide gauge records.

  • Source: GESLA — SEANOE archive
  • Variables: Sea level (hourly)
  • Coverage: ~91 000 station-years from 5 119 gauges
# Path to local GESLA CSV is set in the config
tidal-analysis --config config/tidal_analysis.yaml

FES2014#

Global tidal atlas, finite-element solution.

  • Source: AVISO+ / CNES
  • Resolution: 1/16° (~7 km)
  • Constituents: 34 major and minor
tidal-analysis --config config/tidal_analysis.yaml

TPXO9#

Oregon State University tidal prediction model.

  • Source: OTPS / OSU
  • Resolution: 1/30° (~3.7 km)
  • Constituents: 15 primary + 2 long-period
tidal-analysis --config config/tidal_analysis.yaml

Atmospheric Forcing#

ERA5#

ECMWF fifth-generation global reanalysis used as the baseline for bias correction of CMIP6 atmospheric forcing fields.

  • Source: ECMWF ERA5 — Copernicus CDS
  • Resolution: 0.25°, hourly
  • Variables: 2 m temperature, precipitation, radiation, wind, pressure
# ERA5 files referenced directly in the bias-correction config
bc correct --config config/bc.yaml

Accessing Copernicus Marine Datasets#

The data-loader command wraps the copernicusmarine Python package. A free account is required; authenticate once with copernicusmarine login.

# List variables and metadata for any CMEMS dataset
data-loader --cmems-describe cmems_obs-sst_atl_phy_nrt_l3s_P1D-m

# Browse the full CMEMS catalog
data-loader --copernicus cmems_obs-sst_atl_phy_nrt_l3s_P1D-m \
    --variable sea_surface_temperature \
    --domain -5. 15. 50. 65. \
    --time-range 2023-01-01 2023-01-31 \
    --output sst_jan2023.nc
